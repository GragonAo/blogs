# Blogs 项目中的用户接口说明  
  
## 完善功能  
  
### 接口用户接口权限验证。
- **URL**: `/api/users/<int:pk>/avatar/upload/`  
- **请求方法**: `POST`  
- **描述**: 允许用户上传头像图片，并返回上传后的头像URL。

### 接口获取用户头像。
- **URL**: `/static/avatar/<string:pk>`  
- **请求方法**: `GET`  
- **描述**: 获取用户头像
